<!-- 
vue+webpack的练手项目
 -->
 1. 安装项目依赖
 	$ npm i
 2. 编译
	webpack.config.js所在目录执行  $ webpack 或 $ webpack -w
 3. 访问
  	webpack.config.js所在目录执行  $ webpack-dev-server
    http://localhost:8080/
